# Quentin Bellanger

My personal website: https://quentin-bellanger.com.

![Quentin Bellanger's website](/static/og.png)

## Development

``` bash
# Install dependencies
$ npm install

# Run local server
$ npm run dev

# Generate static project
$ npm run generate

# Publish it on Surge
$ npm run deploy
```
